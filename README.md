This is a News Application built using React.js that displays articles using mock (fake) API data. The app allows users to browse, search, and filter news by categories and language.

⚙️ Features
📰 Display news using mock API data
🔍 Search functionality
🌍 Language filter (Hindi/English)
📂 Category-based filtering
⚡ Responsive UI
🔄 Dynamic updates using React hooks

🛠️ Tech Stack
React.js
JavaScript (ES6+)
CSS
Mock API 

Simulated API behavior using mock data to practice frontend data handling and UI logic.
